# PatrolPP

## 系統環境需求
### PatrolPP 環境需求
* Windows 作業系統
* 所有裝置 (Windows、Mac、Mobile Device) 必須在同一個區域網路環境之下

### Android 環境環境需求
* Windows 設定 Android Debug Bridge(adb)環境變數
### iOS 測試環境需求
* Mac OS 作業系統
* 安裝 XCode 9.0以上
* 安裝 .Net core 2.0
### 瀏覽器測試環境需求
* 下載瀏覽器 webdriver
